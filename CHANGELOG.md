# Changelog

## 0.2.0

- 项目调整为 Linux-only 网络系统学习环境。
- 增加 C Socket、select、poll、epoll、AF_PACKET、TUN、sendfile 和 io_uring 实验。
- 增加 Ethernet、ARP、IPv4、ICMP、UDP、TCP 数据包解析器。
- 增加 network namespace、veth、bridge、路由、NAT 和 netem 脚本。
- 增加 eBPF/bpftrace 网络观测脚本。
- 重写安装、实验、故障排查与数据包路径文档。

## 0.1.0

- Python TCP/UDP、长度前缀、asyncio 聊天室、局域网发现、HTTP/TLS 与诊断工具。
